## Instructions

In the project directory, you can run:

### `npm install` 

To load all dependency.

### `npm start`

Runs the app in the development mode.
Open [http://localhost:3000]

### `npm run build`

Builds the app for production to the `build` folder.

Node version: v14.18.1

npm version: 6.14.15
